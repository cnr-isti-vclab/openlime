Class: LayerAnnotation

LayerAnnotation

LayerAnnotation provides functionality for displaying and managing annotations overlaid on other layers. It supports both local and remote annotation data, selection management, and UI integration.

Features:

  • Display of text, graphics, and glyph annotations
  • Remote annotation loading via JSON/HTTP
  • Selection management
  • Visibility toggling per annotation
  • UI integration with annotation list
  • Annotation event handling

The layer automatically handles:

  • Annotation data loading and parsing
  • UI synchronization
  • Visibility states
  • Selection states
  • Event propagation

Instantiates a LayerAnnotation object.

Parameters:

Name Type Description

The URL of the annotation data (JSON file or HTTP GET Request to an annotation server) or an array of annotations.
Source:
  • LayerAnnotation.js, line 58
Fires:
  • LayerAnnotation#selected - Fired when annotation selection changes,event: with selected annotation as parameter
  • LayerAnnotation#event:loaded - Fired when annotations are loaded
  • Layer#update - Inherited from Layer,event: fired when redraw needed
  • Layer#ready - Inherited from Layer,event: fired when layer is ready
Example
```javascript
// Create annotation layer from remote JSON
const annoLayer = new OpenLIME.LayerAnnotation({
  annotations: 'https://example.com/annotations.json',
  style: '.annotation { color: red; }',
  overlay: true
});

// Listen for selection changes
annoLayer.addEvent('selected', (annotation) => {
  console.log('Selected annotation:', annotation.label);
});

// Add to viewer
viewer.addLayer('annotations', annoLayer);
```
