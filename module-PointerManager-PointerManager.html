Class: PointerManager

PointerManager~ PointerManager

High-level event manager for handling pointer interactions. Captures and classifies PointerEvents (mouse/touch) into higher-level gestures.

new PointerManager(target [, options])

Creates a new PointerManager instance.

Parameters:

Seconds of inactivity before idle event
Source:
  • PointerManager.js, line 51
Fires:
  • PointerManager#event:fingerHover - When pointer moves over target
  • PointerManager#event:fingerSingleTap - On quick press/touch
  • PointerManager#event:fingerDoubleTap - On two quick presses/touches
  • PointerManager#event:fingerHold - When press/touch held for >600ms
  • PointerManager#event:mouseWheel - On mouse wheel rotation
  • PointerManager#event:panStart - When pan gesture begins
  • PointerManager#event:panMove - During pan gesture
  • PointerManager#event:panEnd - When pan gesture ends
  • PointerManager#event:pinchStart - When pinch gesture begins
  • PointerManager#event:pinchMove - During pinch gesture
  • PointerManager#event:pinchEnd - When pinch gesture ends
Example
// Create manager and attach event handler
const canvas = document.querySelector('canvas');
const pointerManager = new PointerManager(canvas);
const handler = {
    priority: 10,
    fingerSingleTap: (e) => {
        console.log("SINGLE TAP at", e.clientX, e.clientY);
    },
    fingerHold: (e) => {
        console.log("HOLD at", e.clientX, e.clientY);
    }
};
pointerManager.onEvent(handler);
